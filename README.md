# 公司主页web项目：bootstrap+django+ajax！

运行方式：在终端运行指令

>  python manage.py runserver

即可在本地浏览器中预览，<http://127.0.0.1:8000/>

其中的图片和文字信息可换成需要的内容。如购买了域名与服务器，则可以布置在公网。
